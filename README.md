🛒 Amazon Clone (Frontend)

A frontend clone of the Amazon homepage built using HTML, CSS, and JavaScript. This project was created to practice web development fundamentals by recreating the layout and design of a popular e-commerce website.

🚀 Features
Amazon-inspired homepage layout
Navigation bar
Search bar
Hero banner
Product sections
Product cards
Footer
🛠️ Technologies Used
HTML5
CSS3
JavaScript
